## 사용 방법

1. 아래 `LSA_v0.3.8.zip` 다운로드
2. 압축 해제
3. LoL 클라이언트 실행
4. `LSA.exe` 더블클릭
5. 완료

## 단축키
- `Ctrl+Shift+O`: 오버레이 표시/숨김
- `Ctrl+Shift+C`: 클릭 통과 모드 토글
- `Ctrl+Shift+P`: [DEV] Mock Phase 전환

## 포함 파일
- `LSA.exe`: 메인 프로그램 (self-contained)
- `data/`: 증강/아이템/챔피언 데이터
